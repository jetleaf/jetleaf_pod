Holder for a constructor argument value, including its optional type and name metadata.

Extends ConvertibleValue to support constructor argument conversion and provides additional metadata for argument resolution.

Example usage:

final vh = ArgumentValue('localhost', 'dart.core.String', 'host');
print(vh.value); // 'localhost'
print(vh.type);  // 'String'

value: The argument value qualifiedName: Required qualified name of the class that the argument is for (The return type of the argument) name: Optional name for the argument
